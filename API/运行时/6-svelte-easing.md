# svelte/easing

缓动功能可指定随时间变化的速率，在使用 Svelte 的内置过渡和动画以及补间和 spring 实用程序时非常有用。svelte/easing 包含 31 个名为出口，一个 linear 轻松和 10 种不同的缓动函数 3 个变种：in，out 和 inOut。

您可以探索使用各种 EASES 轻松 Visuailiser 在示例部分。
